About Coloured Squares

Coloured Squares is a social network, art project and web controller. This version was created from scratch for the A List Apart 10kb challenge.

Instructions

You've been assigned a colour (or an id of three numbers between 0 and 256. Yours is {{req.session.colour.red}},{{req.session.colour.green}},{{req.session.colour.blue}}). Each colour has a squarefield of 16 squares associated with it. You can change the colour value of these squares (also stamping it with your colour and the time). You can also switch your colour (if the colour is not already taken). A copy mode is available to quickly take extract colours. You can transport between squarefields and go back to your home anytime with other buttons. How it is used is completely up to you.

API

  • Pass a "format=JSON" parameter to the squarefield URL to get the squarefield as JSON
  • HTTP POST with values for "square" (1-16) and "red" "green" and "blue" values to light a square.
